Quickstart — Load CSV & Inspect Instantly

JAnalytics is lightweight and straightforward. Instantly load and inspect dataframes by printing them directly with System.out.println(df). No helpers or magic needed.

Java Example
import JData.JAnalytics;
public class Main {
public static void main(String[] args)
{ JAnalytics df = new JAnalyt // Quick inspect — prints formatted table
System.out.println(df);
}
}
Instant introspection

Thanks to JAnalytics.toString() printing a human-friendly tabular view, you can iterate quickly without extra tooling.

API Reference — Core Methods

readCSV(String filename)

Load CSV files. The first row is parsed as column headers. Ignores blank or empty lines. Use toCSV to export.

Parameters
  • filename (String) - path to CSV file
Returns
void - updates dataframe in place; on chaining, returns new copies.

Printing / Inspection

Printing the dataframe directly with System.out.println(df) produces a human-friendly table with column headers, separator line, and rows sized by content width.

Example
JAnalytics df = new JAnalytics();
df.readCSV("people.csv");
System.out.println(df); // nic
toString()

Plotting — PlotBuilder

PlotBuilder lets you render data visualizations like scatter, line, bar, histogram, and pie charts. It opens a Java Swing window without external dependencies.

Scatter / Line
df.plot("x", "y").title("My plot").line(true).show();
Histogram
df.hist("price").show();
Pie / Bar
df.plot("cat","value").pie().show();
Note: Plot windows require GUI environment. For servers, export data for external plotting.

Predictors — Machine Learning Utilities

JAnalytics includes simple train-and-predict models:

  • LinearRegressionModel for least squares regression.
  • SVMModel basic linear SVM, limited single feature.
  • SimpleNN tiny single-layer neural net regressor.
  • DecisionTree recursive split tree for regression.
  • KNN k-nearest neighbors.
  • NaiveBayes frequency based classifier/regressor.
Example — Linear regression on dataset columns
JAnalytics df = new JAnalytics();
df.readCSV("sales.csv");
JAnalytics.Predictors preds = new JAnalytics.Predictors(df);
JAnalytics.Predictors.LinearRegressionModel lr = preds.new LinearRegressionModel();
lr.train("revenue", "adSpend", "discount");
double forecast = lr.predict(1200.0, 5.0);
Models operate on numeric columns getColumnAsDouble. They are ideal for prototyping and exporting trained parameters.

Generative Model — Built-in GPT AI

JAnalytics uniquely integrates a proprietary GPT-based semantic question answering system trained on your own dataset. Ask complex questions naturally and get concise or detailed answers instantly — no external services required.

Main features:

  • Training using TF-IDF and dense embeddings with stemming and stop-word filtering.
  • Similarity-based search combining cosine and Jaccard metrics for high relevance.
  • Configurable epochs, learning rate, and result rank cutoff.
  • Supports different verbosity styles: concise, balanced, detailed.
  • Local, lightweight, self-contained — no cloud or external API needed.
Example — Ask your dataset a question:
GenerativeModel gpt = df.new GenerativeModel();
gpt.trainFromFile("training.txt") // note you need only to train once since the model would cache it if you need to retrain delete the cache file;
gpt.prompt(user prompt);
System.out.println(gpt.answer());

Boost your data workflow by having AI assist with interpretation directly embedded in your Java projects.

Notes & Best Practices

  • For very large CSV files (100MB+), consider streaming or chunked reads to avoid memory issues — readCSV() loads all data at once.
  • Use fillna() or dropna() to handle missing/null data gracefully in pipelines.
  • JAnalytics instances are not thread-safe; clone using deepCopy() before multi-threaded use.
  • Plotting uses Java Swing windows, so requires GUI environment — for headless servers export data via CSV/JSON instead.
  • Predictors included are minimal implementations suitable for prototyping and education; export results for full production use.
  • The GPT generative model operates fully locally, ensuring data privacy with no external calls.
